## 项目介绍

针对当前青少年心理健康问题，结合脑电、肌电等生理信号并通过超现实可穿戴设备，研发基于大语言模型的虚实融合心理检测、预警、疗愈AI系统，打造开源心理教育平台，搭建幸福科技馆，建造公共心理健康体验设施。
